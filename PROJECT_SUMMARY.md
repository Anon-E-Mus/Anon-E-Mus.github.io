## 🎯 Key Technical Features

### Performance
- Canvas-based rendering (60fps target)
- Efficient star update loop
- Smooth mouse interpolation
- Intersection Observer for scroll reveals
- Reduced motion support

### Accessibility
- Semantic HTML structure
- ARIA labels on interactive elements
- Keyboard navigation support
- High contrast mode support
- Focus visible states

### Animations
| Animation | Duration | Easing |
|-----------|----------|--------|
| Preloader | 2.5s | ease |
| Star parallax | Continuous | smooth |
| Fade up (reveal) | 0.8s | ease |
| CTA hover | 0.3s | ease |
| Nebula float | 30s | ease-in-out |
| Character scale | 0.6s | cubic-bezier |

## 🔧 How to Use

1. **Open the website:**
   - Double-click `index.html`
   - Or use Live Server in VS Code

2. **Add your social links:**
   - Open `index.html`
   - Search for `href="#"` in social icon sections
   - Replace with your actual URLs

3. **Customize colors:**
   - Open `css/style.css`
   - Edit `:root` variables at the top

4. **Adjust star field:**
   - Open `js/main.js`
   - Edit `layerStarCounts` array for density
